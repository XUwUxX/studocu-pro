## Tiếng Việt

Một tiện ích mở rộng nhỏ giúp tối ưu hóa trải nghiệm của bạn trên Studocu, hỗ trợ xử lý tài liệu và tải xuống nhanh chóng.

### Tính năng chính
*   **Auto Bypass:** Tự động bỏ qua các bước chặn, trang chờ hoặc yêu cầu khảo sát rườm rà từ hệ thống.
*   **Download Files:** Kích hoạt nút tải xuống trực tiếp các file tài liệu mà không cần tài khoản Premium hoặc phải upload tài liệu khác để đổi điểm.

### Hướng dẫn cài đặt thủ công (Developer Mode)
Vì extension này chưa được cập nhật lên Chrome Web Store, bạn có thể cài đặt trực tiếp qua các bước sau:

1.  Tải mã nguồn của repository này về máy (Bấm vào nút **Code** > chọn **Download ZIP** và giải nén ra một thư mục).
2.  Mở trình duyệt Chrome (hoặc Edge, Brave, Cốc Cốc).
3.  Truy cập vào trang quản lý extension bằng cách gõ đường dẫn: `chrome://extensions/`
4.  Bật chế độ **Developer mode** (Chế độ nhà phát triển) ở góc trên bên phải màn hình.
5.  Nhấp vào nút **Load unpacked** (Tải tiện ích đã giải nén) ở góc trên bên trái.
6.  Chọn thư mục mà bạn vừa giải nén ở bước 1. Extension sẽ xuất hiện và sẵn sàng sử dụng.

### Cách sử dụng
*   Ghim (Pin) extension lên thanh công cụ để dễ theo dõi.
*   Truy cập trang Studocu, tiện ích sẽ tự động chạy ngầm và kích hoạt các tính năng khi bạn mở một tài liệu bất kỳ.

## English

A lightweight browser extension designed to enhance your experience on Studocu, enabling seamless document viewing and fast downloads.

### Core Features
*   **Auto Bypass:** Automatically skips paywalls, pop-up restrictions, and forced surveys.
*   **Download Files:** Unlocks the direct download feature for documents without requiring a Premium subscription or document upload trades.

### Installation Guide (Developer Mode)
Since this extension is not currently hosted on the Chrome Web Store, you can install it manually by following these steps:

1.  Download the source code to your computer (Click the **Code** button > select **Download ZIP** and extract it into a folder).
2.  Open your browser (Chrome, Edge, Brave, or any Chromium-based browser).
3.  Navigate to the extension management page by typing: `chrome://extensions/`
4.  Enable **Developer mode** via the toggle switch in the top-right corner.
5.  Click the **Load unpacked** button located in the top-left corner.
6.  Select the folder where you extracted the source code in Step 1. The extension is now successfully installed.

### How to Use
*   Pin the extension to your browser toolbar for quick access.
*   Go to Studocu; the extension will run automatically in the background and activate its tools whenever you view a document.
